Prioritizing Pump and Compressor Maintenance Schedules

This was a consulting job I did for an oil refinery I'd worked with previously. We used small internet-of-things (IoT) sensors to monitor vibration levels and cumulative run time of pumps and compressors at various refinery units. The data was logged to a Snowflake SQL database hosted on AWS. I configured a PowerBI Snowflake connector to query database diffs. Using planned unit downtime schedules, and considering things such as max vibration, median vibration, and cumulative run time, I developed a dashboard that would prioritize maintenance based on scheduled unit shutdowns.
